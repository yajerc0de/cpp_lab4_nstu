Выполнить задание по варианту с использованием алгоритмов генерации mt19937_64 

                            Вариант 12 
1. Напишите программу, в которой определены массивы arr1 и arr2, состоящие из n элементов (n ≥ 10),
инициализируйте массивы случайными целыми числами из диапазона [10, 100]. 
2. Выведите элементы массивов из пункта 1, которые являются простыми числами из диапазона [10, 70]. 
3. Отсортируйте массив arr1 по убыванию, а массив arr2 по возрастанию. Выведите полученные массивы. 
4. Поменяйте местами элементы массива arr1, имеющие нечётные индексы, с элементами массива arr2, 
имеющими чётные индексы. Выведите полученные массивы. 
5. Создайте массив из n чисел (n ≥ 15), состоящий из случайных целых чисел из диапазона [10000, 20000].
Определите числа, у которых совпадают как минимум две цифры. Создайте новый массив и заполните его этими
числами.
